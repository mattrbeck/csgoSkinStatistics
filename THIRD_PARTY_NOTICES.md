## Blue-gem pattern percentages (`blue-gem.json`)

`blue-gem.json` is generated by `scripts/update_bluegem.js` from the **`data/bluegem.json`** file
in the [csfloat/extension](https://github.com/csfloat/extension) repository, which is MIT-licensed.
We reshape it (extract the playside/backside blue percentages, key it by weapon/pattern) but the
underlying measurements are CSFloat's published data. Per CSFloat's own generator, those numbers
originate from the **CSBlueGem** ([bluegem.app](https://bluegem.app)) pattern analysis — thanks to
both projects.

We use only the openly-published, MIT-licensed GitHub data file; we do **not** access csfloat.com's
live service or API.
